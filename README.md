# Andrew Mock

Hey, I'm Andrew. I'm a software development senior at Brigham Young University-Idaho with a growing interest in cybersecurity, web development, and building useful software.

I enjoy learning how things work, figuring out why they break, and finding practical ways to make them better. That applies to software, computers, cars, and pretty much anything mechanical.

## What I'm Working On

- Building responsive websites for local service businesses
- Developing secure Flask applications with contact forms, API integrations, and production-focused security
- Improving my skills with Object Oriented Programming, and cybersecurity
- Experimenting with automation, game-server customization, and personal software projects

## Recent Projects

### 360Epoxy's Business Website
Designed and developed responsive website for an epoxy flooring company. This project included custom service pages, project galleries, lead-generation form, SEO (Search Engine Optimization) improvements, and security-focused Flask backends.

### Chess Project

Developed a playable chess application that recreates the rules and strategy of traditional chess. The project handles piece movement, captures, turn management, and board-state validation.

This project strengthened my understanding of object-oriented programming, game logic, and managing interactions between multiple systems.

### Artillery Simulator

Built an artillery simulator that models projectile movement using launch angle, velocity, gravity, and environmental conditions.

The project demonstrates how physics and mathematics can be applied through software. It improved my experience with simulations, validating user input, and clearly presenting calculated results.

### Lunar Lander

Created a lunar landing simulation where the player must carefully manage fuel, velocity, and altitude to safely land a spacecraft.

The project combines physics-based calculations with real-time decision-making and game-state management. It challenged me to create balanced gameplay while modeling gravity, acceleration, and fuel consumption.
